Stealthy social network (Twitter / Facebook) based pentest drop box

Pentest drop box - a portable device that is plugged (or connected wirelessly) to the target network during pentests and can be controlled from a command and control (C&C) server to perform the following functions: Network monitoring / sniffing Scan /exploit other hosts in the network ARP / DNS poisoning for MiTM attacks Denial of service Drop boxes can call C&C using a number of methods, not limited to: Reverse SSH (over OpenVPN) SSH over HTTP(s) DNS tunneling - here is a good write-up Route through TOR ICMP based shell Egress buster - try all outbound ports until you find an open port Use a 3G dongle so that the C&C commands don’t touch the target network Most of the above methods will require a persistent connection (shell) with your C&C, increasing the chances of detection. ↪
